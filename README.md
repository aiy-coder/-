# 高校教务系统学生端 — 前端 UI 重构方案

AI UI 课程期末综合实训项目。使用纯原生 HTML/CSS/JavaScript 构建的新一代教务系统前端演示总控台。

## ✨ 项目亮点

- **13 个页面**覆盖教务系统全部核心场景（仪表盘、课表、选课、成绩、通知、考试、培养方案、缴费、资源、评教、设置、个人信息）
- **Bento Grid 便当盒布局** — 首页非对称卡片网格，重要信息占据更大视觉权重
- **12 个 ECharts 图表** — GPA 趋势、学分达成、能力雷达、学业健康仪表盘、费用构成等
- **13 个 localStorage 键**跨页面状态管理 — 暗黑模式、未读 Badge、通知已读、用户姓名、选课购物车全站同步
- **暗黑模式**全站 CSS Variables 一键切换
- **四档响应式**（480/768/1024/1280）+ 移动端底部 TabBar + 全局搜索 Ctrl+K
- **选课购物车三态流程** — 加入→提交→取消→退选，localStorage 持久化
- **通知稍后再读+置顶** — 轻量级待办清单交互
- **骨架屏**加载态 + 时段感知问候 + 页面切换过渡动画

## 🚀 快速开始

直接用浏览器打开 `综合整理/login.html`，输入任意学号 + 密码（≥3 位）即可进入系统。

## 🛠 技术栈

| 层 | 技术 | 说明 |
|:---|:---|:---|
| 结构 | HTML5 | 语义化标签、Bento Grid 布局 |
| 样式 | CSS3 | Flexbox/Grid、CSS Variables、backdrop-filter、cubic-bezier 动效、@media 响应式 |
| 逻辑 | Vanilla JS | DOM 操作、事件委托、闭包防抖、localStorage 状态管理、动态渲染 |
| 图表 | ECharts 5 | 折线/柱状/饼图/雷达/仪表盘，共 12 个实例 |
| 图标 | Lucide | 线性图标库，全站 stroke-width: 1.75 |
| 字体 | Inter + PingFang SC | 英文/数字 + 中文，零框架依赖 |
